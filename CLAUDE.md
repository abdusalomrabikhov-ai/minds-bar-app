# TeamTask — project instructions

## Session-end logging

Before ending a session:
- If a deploy happened (`railway up`, `./deploy.sh`), append an entry to `DEPLOY.md` under a "Deploy log" section: date, what changed, result.
- If code changes/additions/updates happened, append a short entry to `MEMORY.md`: date, what changed, why (skip trivial edits like typo fixes).

Keep entries terse — one bullet per change, newest at top of each log section.
