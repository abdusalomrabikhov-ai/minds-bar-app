# TeamTask — Инструкция по запуску

## Быстрый старт (уже готово на вашем компьютере)

```bash
cd ~/teamtask
npm start
```

Откройте браузер: **http://localhost:3000**

Логин: `admin@teamtask.com`  
Пароль: `admin123`

---

## Первые шаги

1. **Войдите** как администратор
2. **Измените пароль** в разделе Настройки
3. **Добавьте проекты** (ваши 16 клиентов) через кнопку "+" в боковом меню
4. **Добавьте сотрудников** → Команда → "+ Добавить сотрудника"
5. **Создавайте задачи** через кнопку "Новая задача" в правом верхнем углу

---

## Подключение Telegram бота (уведомления)

1. Зайдите на https://t.me/BotFather в Telegram
2. Напишите `/newbot` и следуйте инструкциям
3. Получите токен вида: `7123456789:AAHxxxxxxxxxxxxxxx`
4. Создайте файл `.env` в папке teamtask:

```
PORT=3000
JWT_SECRET=любой-случайный-текст-50-символов
TELEGRAM_BOT_TOKEN=7123456789:AAHxxxxxxxxxxxxxxx
TELEGRAM_BOT_USERNAME=имя_вашего_бота
```

5. Перезапустите: `npm start`
6. Каждый сотрудник заходит в Настройки → "Подключить Telegram" → получает код → отправляет боту

## Деплой в интернет (опционально)

### Вариант 1: Railway (бесплатно)
1. Зарегистрируйтесь на https://railway.app
2. Подключите GitHub репозиторий или загрузите папку
3. Railway автоматически запустит `npm start`
4. Добавьте переменные окружения из `.env` в настройках Railway

### Вариант 2: Render (бесплатно)
1. Зарегистрируйтесь на https://render.com
2. New → Web Service → подключите репозиторий
3. Build Command: `npm install`
4. Start Command: `npm start`

## Роли в системе

| Роль | Возможности |
|------|------------|
| Администратор | Видит все задачи, создаёт/удаляет задачи, управляет командой и проектами |
| Сотрудник | Видит только свои задачи, меняет статус, оставляет комментарии |

---

## Статусы задач

- 🔵 **Новая** — задача создана
- 🟡 **В работе** — сотрудник начал выполнение
- 🟢 **Готово** — задача завершена

## Уведомления

Сотрудники получают уведомления:
- При назначении новой задачи (Telegram + браузер)
- За 24 часа до дедлайна (Telegram + браузер)
- За 1 час до дедлайна (Telegram + браузер)
